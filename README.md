# Persona 5 | Exercícios

Projeto de exercícios de HTML e CSS com estética inspirada em **Persona 5** — menu animado com Three.js, partículas, vídeo de fundo e navegação estilizada.

## Como rodar

Clone o repositório e abra o `index.html` com o **Live Server**.

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

## Exercícios

| Exercício | Descrição |
|-----------|-----------|
| **Ex 1** | Cartão de visita — Ren Amamiya / Joker |
| **Ex 2** | Lista de personas, habilidades e itens |
| **Ex 3** | Cards de destinos icônicos de Tóquio |
| **Ex 4** | Loja fictícia dos Phantom Thieves |

## Stack

- HTML5 / CSS3
- [Three.js r128](https://threejs.org/) — vídeo 3D no fundo
- [GSAP 3](https://gsap.com/) — animações e partículas

## Preview

Menu principal com vídeo de fundo, efeito parallax com o mouse, partículas na seleção e trilha sonora.
