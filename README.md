# Persona 5 | Exercícios

Exercícios de HTML e CSS com estética do Persona 5. Menu animado com Three.js, vídeo de fundo, partículas e trilha sonora.

## Como rodar

Clone o repositório e abra o `index.html` com o Live Server.

## Exercícios

**Ex 1**: Cartão de visita do Ren Amamiya / Joker

**Ex 2**: Lista de personas, habilidades e itens

**Ex 3**: Cards de destinos icônicos de Tóquio

**Ex 4**: Loja fictícia dos Phantom Thieves

## Stack

Three.js r128 para o vídeo 3D no fundo e GSAP 3 para as animações e partículas.
